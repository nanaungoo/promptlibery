# Study Notes Prompt Library (လေ့လာမှုမှတ်စုနှင့် Prompt များ)

Personal repository for structured AI prompts, workflows, configurations, and learning paths.
AI prompt များ၊ လုပ်ငန်းစဉ်များ၊ ပတ်ဝန်းကျင်ပြင်ဆင်မှုများနှင့် လေ့လာမှုလမ်းညွှန်များကို စုစည်းထားသော ကိုယ်ပိုင်မှတ်စုစုစည်းမှု။

## Structure (ဖွဲ့စည်းပုံ)

### `config_setting/` (IDE Settings)
*   `zde/zed_setting.md`: Zed config notes. (Zed config မှတ်စုများ)
*   `zde/discussion_noteWithai.md`: AI discussion notes on config. (Config ဆိုင်ရာ AI နှင့်ဆွေးနွေးချက်များ)

### `vibe_coding/` (Agentic Tools Config & Rules)
Instructions and rules for AI editors (Aider, Continue, Cursor, Windsurf, Zed AI, Neovim Avante, Open Interpreter, OpenHands).
AI tools များအတွက် စည်းမျဉ်းများနှင့် လမ်းညွှန်ချက်များ (Aider, Continue, Cursor, Windsurf, Zed AI, Neovim Avante, Open Interpreter, OpenHands)။

### `Prompt_examples/` (Specialized Prompts / အထူးပြု prompt များ)
*   `Codebase_Reverse_Engineering_prompt.txt`: Reverse engineering codebase. (ကုဒ်များကို လေ့လာဆန်းစစ်ရန် prompt)
*   `Universal_Refactor_Master.txt`: Universal code refactor workflow. (ကုဒ်များပြန်လည်ပြုပြင်ရန် prompt)
*   `context_collection_prompt_for_aiIDE.md`: Context collection for AI IDEs. (AI IDE များအတွက် context စုဆောင်းရန် prompt)
*   `lerning_path_create_prompt_for_claw.md`: Study path generator with Claude. (Claude ဖြင့် လေ့လာမှုလမ်းကြောင်းဖန်တီးရန် prompt)
*   `master_vibe_pro.txt`: Core vibe coding logic. (ပင်မ vibe coding စည်းမျဉ်းများ)
*   `programming_procedure_prompt.txt`: Standard programming procedures. (ကုဒ်ရေးသားခြင်းလုပ်ငန်းစဉ်လမ်းညွှန် prompt)

## Usage (အသုံးပြုပုံ)

Copy and use these prompts with LLMs/agents for coding and study.
ကုဒ်ရေးခြင်းနှင့် လေ့လာခြင်းများအတွက် ဤ prompt များကို ကူးယူပြီး LLMs/agents များတွင် အသုံးပြုပါ။
